## PostgreSQL

Para entrar al cliente de PostgreSQL ejecutamos el siguiente comando:

```console
sudo -i -u postgres psql
```

La configuración de la base de datos en el `settings.py` debe contener también la dirección del _"host"_. Si estamnos usando [prettyconf](https://prettyconf.readthedocs.io/en/latest/) quedaría algo así:

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': config('DB_NAME', default='blog'),
        'USER': config('DB_USERNAME', default='blog'),
        'PASSWORD': config('DB_PASSWORD', default='blog'),
        'HOST': config('DB_HOST', default='localhost')
    }
}
```
