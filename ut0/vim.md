# Vim

[Vim](https://www.vim.org/) (Vi improved) es un **editor de texto para línea de comandos** con infinitas posibilidades.

![Vim Logo](./images/vim-logo.svg)

## Recursos de aprendizaje

- [Tutorial interactivo de vim](https://www.openvim.com/tutorial.html)
- [Listado de comandos](https://vim.rtorr.com/)

## Ejercicio

Partiendo de este fichero [nose-v1.py](./files/nose-v1.py) realiza las modificaciones pertinentes para llegar a este otro fichero [nose-v2.py](./files/nose-v2.py).

Notas:

- Te puedes apoyar en la herramienta `vimdiff` para comprobar diferencias, e incluso para editar sobre la marcha:

  ```console
  vimdiff nose-v1.py nose-v2.py
  ```

- Mantén tus manos sobre el teclado. No uses ni cursores ni ratón. Evita copiar y pegar desde un fichero al otro.
- `:set list` te permitirá dentro de vim visualizar los espacios en blanco y caracteres ocultos.
