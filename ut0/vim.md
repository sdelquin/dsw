# Vim

[Vim](https://www.vim.org/) (Vi improved) es un **editor de texto para línea de comandos** con infinitas posibilidades.

![Vim Logo](./images/vim-logo.svg)

## Recursos de aprendizaje

- [Tutorial interactivo de vim](https://www.openvim.com/tutorial.html)
- [Listado de comandos](https://vim.rtorr.com/)

## Ejercicio

Partiendo de este fichero [nose-v1.py](./files/nose-v1.py) realice las modificaciones pertinentes para llegar a este otro fichero [nose-v2.py](./files/nose-v2.py).

Una vez terminadas las modificaciones puede comprobar que los ficheros son iguales lanzando el siguiente comando en una terminal:

```console
cmp nose-v1.py nose-v2.py && echo '¡IGUALES!'
```
